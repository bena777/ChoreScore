## ChoreScore
*Created by: Benjamin Adelman, Haden Anderson, Mason Kolb, & Hudson Whitney*

### Temp | For Devs:

Install dependencies:
```bash
npm install
```

Run frontend (dev build, will auto-refresh):

In a new terminal,
```bash
cd client
npm run dev
```
* Open http://localhost:5173/ on any local web browser

Run backend (dev build, will auto-refresh):

In a new terminal,
```bash
cd server
npm run dev
```
* Open http://localhost:5000/api/ on any local web browser
